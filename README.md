# Fansly Downloader (Extension)

A Chrome Extension that monitors and collects pics/vids as you browse Fansly, allowing you to download them individually or bundled into a ZIP file. 

Instead of manually saving media one by one, this extension logs the network requests of the media you view and then downloads them once you're finished browsing the files you want.

## Features

* **Background Monitoring:** Automatically captures media URLs as they load on the page.
* **HLS Video Support:** Automatically fetches, stitches, and converts `.m3u8` and `.ts` video segments into standard downloadable video files.
* **Smart Deduplication:** Identifies and filters out lower-quality duplicate images (e.g., watermarked variants or thumbnails) generated by the CDN at the same timestamp.
* **Quality Control:** Built-in (and toggleable) dimension and file-size checks ensure you only download full-resolution content (skips UI elements and thumbnails under 500px or 100KB).
* **Memory-Safe Zipping:** Uses `client-zip` and the File System Access API to stream large ZIP files directly to your hard drive, preventing browser memory crashes when downloading massive videos.
* **Interactive Popup:** Gives you granular control to download "Images Only," "Videos Only," or "All Media," as well as toggles for monitoring and size restrictions.

## Installation

Since this extension is not listed on the Chrome Web Store, you will need to install it manually via Chrome's Developer Mode.

1. **Download the code:** * Click the green **Code** button at the top of this repository and select **Download ZIP**. 
   * Extract the ZIP file to a folder on your computer.
2. **Open Chrome Extensions:**
   * Open Google Chrome and type `chrome://extensions/` into the address bar, or go to the three-dot menu > **Extensions** > **Manage Extensions**.
3. **Enable Developer Mode:**
   * Toggle the **Developer mode** switch in the top right corner of the page.
4. **Load the Extension:**
   * Click the **Load unpacked** button that appears in the top left.
   * Select the folder where you extracted the extension files.
5. **Pin it:**
   * Click the "puzzle piece" icon in your Chrome toolbar and click the pin icon next to "Fansly Image Collector" so it's easily accessible.

## How to Use

1. **Browse:** Navigate to Fansly and start browsing posts or messages. You will see a small badge number appear on the extension icon indicating how many media files have been captured.
2. **Open the Menu:** Click the extension icon to open the control popup.
3. **Download:** Click your preferred download option (All, Images, or Videos). 
   * *Note: If downloading multiple files, Chrome will prompt you for a save location to build the ZIP file.*
4. **Clear:** The queue will automatically clear upon a successful download. You can also manually wipe the queue at any time using the "Clear List" button.
